Este sistema realiza a importação de arquivo de observação no formato XLSX e aplica o método Sec-UP-Free (Sequential and Combinatorial Geometry-Free Identification of Unstable Points for Geodetic Deformation Monitoring). O método combina a análise combinatória e teste de razão de verossimilhança em um procedimento sequential. Ele utiliza as diferenças nas observações em duas épocas para identificas pontos instáveis no monitoramento geodésico de estruturas, utilizando simulação Monte Carlo para o valor crítico.
